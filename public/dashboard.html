Інфраструктура для MQTT та Node-RED

Цей сервіс піднімає MQTT broker, Node-RED та рольовий веб-доступ. На цій сторінці є короткий опис проєкту, а самі Node-RED dashboards доступні лише для окремої ролі після входу.

◐

MQTT broker

TCP і WebSocket endpoint для пристроїв, автоматизації та браузерних клієнтів.

▣

Node-RED

Редактор і flow-логіка захищені роллю адміністратора.

◉

Dashboard access

Окрема роль тільки для перегляду Node-RED dashboards без доступу до адміністративних екранів.

Рівні доступу

Гість і звичайний користувач бачать лише цю сторінку. Переглядачі відкривають Node-RED dashboards, а решта доступу залишається за адміністратором.

Public Опис сервісу та посилання на проєкт.
Viewer Тільки перегляд Node-RED dashboards.
Admin Повний доступ до editor, адміністративної панелі та керування ролями.
